APP DE SORTEO

Funcionamiento:

1. Los usuarios introducen sus datos.
2. Los datos se guardan en el navegador.
3. El administrador puede:
   - Descargar todos los participantes en un archivo TXT
   - Realizar el sorteo

Archivos incluidos:

index.html -> formulario para participantes
admin.html -> panel administrador
style.css -> estilos
script.js -> lógica
